Install Docker Desktop
Executez les commandes suivantes dans un terminal command windows:
cd YOUR_PATH_TO_PROJECT/pipeline
docker-compose up airflow-init

Executez les commandes suivantes dans un autre terminal commande windows:
cd YOUR_PATH_TO_PROJECT/pipeline
docker-compose up -dans

Se connecter a Airflow:
utilise l'url: localhost:8080  (peut prendre quelque temps a s'initialiser)
